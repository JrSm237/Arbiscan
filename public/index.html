Configuration Telegram
1
Ouvre Telegram et cherche @BotFather. Envoie /newbot et suis les instructions. Tu recevras un token de la forme 123456:ABC-xxxx.
2
Crée un groupe Telegram (ou utilise ton compte perso). Ajoute ton bot au groupe. Pour récupérer le Chat ID du groupe : cherche @userinfobot et transfère-lui un message du groupe. Tu verras l'ID (commence par -100 pour les groupes).
3
Dans Render → ton service → Environment, ajoute ces variables :
TELEGRAM_BOT_TOKEN = ton token
TELEGRAM_CHAT_ID = ton chat ID
ALERT_SPREAD = 2.0 (seuil en %)
SCAN_INTERVAL = 60000 (60 secondes)
4
Teste la connexion avec le bouton ci-dessous. Si ça fonctionne, tu recevras un message de confirmation sur Telegram.
